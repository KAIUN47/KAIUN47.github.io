## 前言

日本找工作有多麻烦，大家多多少少都有听过，具体可以百度谷歌看看。本文主要用于记录我在日本找工作的心酸历程（还没发生）。也许对在日本留学后想留在日本工作的同学有帮助。





## 为什么要在日本工作

询问这个问题后，和我一起留学的一些朋友纷纷表示毕业后会回国找工作。虽然我不想知道他们为什么这么做，但我害怕这会动摇我留在日本的决心。至于我自己的原因，虽然可能有些肤浅，但已经足够了。

### 收入

刚入职到手大概1万人民币，起点比较高。每年享有带薪休假，加班补贴和夜班津贴等福利。此外，薪资还有每年的涨幅，且有奖金、交通补贴和住宿补贴等其他福利。

### 医保

深刻体会到了日本医疗保险制度的完善。特别是经历了国内亲人患了癌症，把一辈子辛辛苦苦赚来的钱全部用来看病之后。医疗费用报销 70%，报销范围之大令人震惊。之前去洗个牙也算入了报销的范围内，看皮肤上的痘痘、普通的感冒、喉咙发炎等，看病费和药费全都在报销范围内。看病的时候医生也会非常耐心地告诉我患病的原因、推荐的治疗方法，以及开完药后仔细解释为什么开这个药、可能存在的副作用等。真正地感受到了我确实是被当作一个病人。

自己由于没有患过重大疾病，对日本高超的医疗水平以及重大疾病每个月个人支付的上限制度等并不是很清楚。当然，希望自己永远不会用到这个制度。

### 治安

全东京治安最差的地方是新宿区[^1]，我曾在那里居住了一年，并在唐吉可德店里工作了半年的夜班，直到凌晨两三点钟。在那里经历的比较印象深刻的事情不是很多：

- 有一次，一对日本夫妻在店里吵架，甚至拿店里的东西砸对方。后来我们报了警。
- 有一个日本女孩因为和男朋友分手，在店里躺在地上，用自己的指甲划自己的大腿，甚至划出了血。我们报了救护车，急救人员看到她状态还不错，就把她绑到担架上。
- 在深夜寂寞的时候，有些可爱的日本小姐姐会跑来问我们：“请问这里有卖震动棒吗？”看完之后还会继续追问：“有没有更大的？”
- 下班回家的时候，时不时会遇到几个人在路边喝酒。那时候我还不太懂得和他们交流，但最近我也开始喜欢在路边和其他日本人一起喝酒，我能理解他们了。按照日本人的脾气，也不会发生什么大事。下次我有空，可以去便利店买罐酒加入他们，这是夜晚！

### 房价

我不太关注房价，反正这里比上海便宜。将来买房子的压力也不会太大，也许可能甚至不需要买房。世界上很少有城市的房价能比上海更高，难道就没有人来救一下中国的房价吗？

### 因为是日本

作为一个老二次元死宅，我只想去日本，因为那里是朝圣的天堂。

#### SPI是什么

リクルートが販売している適性検査，SPI能力测试的结果显示了申请人的个性、对工作的适合性以及他们将适合的组织类型[^3]。公司可以利用SPI的结果来

- 参考信息，以帮助你在面试中了解学生的情况
- 人格信息，以帮助你决定该人在加入公司后将被派往何处工作
- 人格信息，以帮助主管在该人加入公司后了解他/她的情况

简单点来说就是判断你这个人有没有基本尝试的一个考试。里面有语言题目（日语和英语），数学题目等。让外国人来做面向日本人的日语题目真的是挺过分的。JLPT的N1题目都没这么把我虐过。就职活动前可能需要买一本SPI3的模拟题做做。

  - 长期的大概几周

### 日本就业分类

| メンバーシップ型               | ジョブ　フォーカス型     |
| ------------------------------ | ------------------------ |
| 首先录取，工作内容录取之后再定 | 看重经验与技能，进行录取 |
| 按照年龄给工资                 | 根据成果拿工资           |
| 毕业生一起录取                 | 每年都有录取             |

- 终身聘用制度

| 総合職(一般都是这个录取)     | 専門職                  |
| ---------------------------- | ----------------------- |
| 各种地方工作                 | 活用自己的专业知识      |
| 营业销售等人事经理管理等职位 | 工程师设计师研究者等    |
| ジョブローテーション　○      | ジョブローテーション　△ |

根据录取文化的攻略法

| メンバーシップ型     | 协调性：团队交流，企业价值观的理解等 |
| -------------------- | ------------------------------------ |
| 终身聘用制度         | 表达自己长时间工作的意欲             |
| ジョブローテーション | 比起技能更重视潜力                   |

关于理科学生的采用

| 本科生 | 修士             | 博士   |
| ------ | ---------------- | ------ |
| 総合職 | 総合職 or 専門職 | 専門職 |

- 在日本年龄越大，毕业生采用的企业越少
- 更具年龄和技能，推荐选择「キャリア採用」（面向有社会经验的人的录取）
